# Assets Directory

This directory should contain the following image files for your app:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon displayed on home screen
   - Should be square with no transparency

2. **splash.png** (1242x2436 for iOS, 1080x1920 for Android)
   - Splash screen shown when app launches
   - Should match your app's branding

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Should be centered on transparent or colored background

4. **favicon.png** (48x48)
   - Web favicon (optional for mobile)

## Quick Setup

You can use online tools to generate these:
- [App Icon Generator](https://www.appicon.co/)
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)

Or create them manually:
- Use design tools like Figma, Photoshop, or Canva
- Export as PNG with the specified dimensions
- Place them in this `assets/` directory

## Temporary Placeholder

For development, you can temporarily remove the asset references from `app.json` or use a solid color background until you create the actual assets.
